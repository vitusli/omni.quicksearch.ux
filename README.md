# Quick Search UX

Unified quick-search extension for Isaac Sim.

## What it does

- Registers one quick-search provider that includes:
  - Menu bar actions (from `omni.menu.quicksearch` behavior)
  - Stage create/add and physics actions (from `omni.stage.quicksearch` behavior)
- Opens with `Ctrl+F`.
- Keeps Stage arrow navigation hotkeys (`Left`, `Right`, `Down`) in the Stage window.

## Usage

1. Add the parent folder of this extension as an extension search path in Isaac Sim.
2. Enable **Quick Search UX** in `Window > Extensions`.
3. Press `Ctrl+F`.
4. Search for menu entries, create/add actions, and stage-related actions in one place.
